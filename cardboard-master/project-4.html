Product Catalog Application

A full-stack web application designed to display and manage a product catalog efficiently with a modern, responsive interface. The application provides users with the ability to browse, filter, and sort products by categories and price.

The product details include name, description, image, and price, all displayed in a clean and structured layout that enhances the shopping experience.

  • Features:
    • View a wide range of products in a responsive grid layout
    • Filter products by category for quick navigation
    • Search products dynamically by name
    • Sort products based on price (ascending/descending)
    • Backend supports data seeding for categories and products
    • Fully localized interface (Arabic) for regional accessibility
  • Technologies Used:
    • Frontend: React.js, HTML, CSS
    • Backend: Spring Boot (Java)
    • Database: PostgreSQL
    • API: REST API for client-server communication
