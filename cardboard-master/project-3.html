User Management Application

A web application designed to manage user information efficiently through a clean and responsive interface. The app provides core functionalities to add, view, edit, and delete user records. Users can see details such as name, username, and email in a structured table format, making data handling simple and organized.

Features:
  • - Add new users with a simple form
  • - View all users in a dynamic table
  • - Edit and delete user information
  • - Responsive design for a smooth user experience
Technologies Used:
  • Frontend: React.js / HTML / CSS / Bootstrap
  • Backend: Spring Boot (Java)
  • Database: PostgreSQL
  • API: REST API for client-server communication
